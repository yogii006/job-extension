Privacy Policy for Mailcruit

Last updated: February 2026

Mailcruit is a Chrome extension designed to help users generate professional job application emails based on job descriptions provided by the user.

Information We Collect

Mailcruit may collect the following information that users voluntarily provide:

  • Name
  • Email address
  • Phone number
  • Job description text

This information is used solely to generate personalized job application emails.

How We Use Information

  • User profile data is stored locally in the user's browser using Chrome storage.
  • Job description text and profile details may be securely transmitted to third-party AI services (such as OpenAI or Google Gemini) for the sole purpose of generating email drafts.

Mailcruit does not:

  • Sell user data
  • Share user data with advertisers
  • Track browsing activity
  • Collect browsing history

Data Storage

All user profile information is stored locally in the user's browser. Mailcruit does not maintain external databases containing user personal data.

Third-Party Services

Mailcruit integrates with external AI API providers to generate email content. These providers process user-submitted text solely for generating the requested output.

Data Security

All communications with external APIs occur over secure HTTPS connections.

Changes to This Policy

This Privacy Policy may be updated from time to time. Continued use of the extension after updates constitutes acceptance of the revised policy.
