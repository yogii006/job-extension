# Privacy Policy for Mailcruit (Job Application Assistant)

**Effective Date:** September 9, 2025

## Overview
Mailcruit is a Chrome extension that helps users generate professional job application emails using AI (OpenAI or Gemini). We value your privacy and are committed to protecting your personal information.

## What Data We Collect
- **Profile Information:** Name, email, phone, experience, skills, education, LinkedIn, GitHub, portfolio, and resume (if provided by you).
- **API Keys:** Your OpenAI or Gemini API key, stored locally in your browser.
- **Usage Data:** None. We do not track or collect any analytics or usage data.

## How Your Data Is Used
- Your profile and resume are used only to generate personalized job application emails.
- Your API key is used solely to authenticate requests to the selected AI provider (OpenAI or Gemini) for generating emails.
- No data is sent to any server other than the selected AI provider's API endpoint.

## Data Storage
- All data is stored locally in your browser using Chrome's sync and local storage.
- No data is transmitted to or stored on any third-party server except for the purpose of generating emails via OpenAI or Gemini APIs.

## Data Sharing
- We do not share, sell, or transmit your data to any third party, except as required to generate emails using the selected AI provider.

## Security
- Your API keys and profile data are stored securely in Chrome's storage.
- We recommend never sharing your API keys with anyone.

## User Control
- You can view, update, or delete your profile and API keys at any time via the extension's settings page.
- You can remove the extension at any time, which will delete all stored data from your browser.

## Changes to This Policy
We may update this privacy policy from time to time. Any changes will be posted within the extension or on the extension's listing page.

## Contact
For questions or concerns about your privacy, please contact the developer via the Chrome Web Store support page.
